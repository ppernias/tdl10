# TDL Paper

**TDL (Tutor Description Language) - Paper on context engineering for educational AI tutors**

## Overview

This repository contains the research paper exploring how context engineering techniques, implemented through ADL (Assistant Description Language) and its educational adaptation TDL, can be used to create personalized educational tutors.

- **Pedro Pernías Peco** - Lead Author - Universidad de Alicante
